# Photoshop Файзуллин Артур

Небольшой редактор изображений на React + Vite. Поддерживает загрузку PNG/JPG/GB7, просмотр каналов, пипетку с RGB/LAB, уровни, масштабирование изображения, фильтрацию через ядра 3x3 и экспорт в PNG/JPG/GB7.

## Запущенный проект

Проект доступен по ссылке:

```text
https://photoshop-steelba11s.onrender.com/
```

## Локальный запуск

Установить зависимости:

```bash
npm install
```

Запустить проект в режиме разработки:

```bash
npm run dev
```

После запуска Vite откроет приложение на локальном адресе, обычно:

```text
http://localhost:5173
```

## Сборка и проверка

Проверить TypeScript:

```bash
npm run typecheck
```

Запустить тесты:

```bash
npm test
```

Собрать production-версию:

```bash
npm run build
```

Собранные файлы появятся в папке `dist`.

## Локальный preview production-сборки

После сборки можно запустить preview:

```bash
npm run preview
```

## Фильтрация

Инструмент "Фильтры" открывает модальное окно с предпросмотром, выбором пресета, сеткой ядра 3x3, выбором каналов и стратегией обработки края. Доступны тождественное отображение, повышение резкости, фильтр Гаусса 3x3, прямоугольное размытие, два ядра оператора Прюитта и медианная фильтрация 3x3.

## Формат GB7

GB7 хранит 7-битное grayscale-изображение и опциональную маску прозрачности. При импорте GB7 можно оставить файл в native grayscale-режиме или конвертировать его в RGBA для обычной работы редактора.
